Installation Steps

Make sure Python 3 is installed in your system. Check using
"python --version"

Clone the repo using git:
git clone https://github.com/anburocky3/tamil-mp3-downloader

cd tamil-mp3-downloader

The project has a requirements.txt:
pip install -r requirements.txt

This involves a command:
python main.py

Files are saved to: output/CategoryName/AlbumName/FolderName
